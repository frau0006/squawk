My name's Cole! Some people call me Spekk, too. :) And I says, says I, just use Windows or macOS. It just werks. Linux was never a good destop environment. It's passable as a Facebook machine. Trust me bro it isn't worth it using it as your daily driver unless you're far gone schizo afraid of the botnet or you run PC hardware that's like 13 years old and Linux is the only solution for security updates.

VIM too. Learn VIM all you wish. Learn the keybinds, get gud at it. But don't get pressured into using it as your everthing machine. Dude it's [Current Year] stop larping like you're at Bell Labs in the 70s writing primitive C on a PDP-11 over a dumb-terminal. Just use vsCode or Atom or something idk lol. It's taken long enough to realize but if you're a total n00b then trying to be some like, cryptic code wizard because you do everything in a plain-text terminal is silly. Get gud at being a code normie before you try to be a code wizard. You should still learn VIM though, it's just an a-OK skill.
